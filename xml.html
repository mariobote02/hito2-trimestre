XML

Historia XML

XML fue desarrollado por el World Wide Web Consortium (W3C) a finales de la década de 1990.

Su objetivo principal es proporcionar una forma de estructurar datos de manera legible tanto para humanos como para máquinas.

Se inspiró en el lenguaje de marcado Standard Generalized Markup Language (SGML)

Normas de uso y sintaxis

XML se utiliza para representar datos en un formato estructurado, utilizando etiquetas para definir los elementos y atributos para proporcionar información adicional sobre esos elementos.

La sintaxis básica de XML incluye elementos, atributos, comentarios y procesamiento de instrucciones.

Los documentos XML deben seguir ciertas reglas, como tener un elemento raíz único y utilizar etiquetas correctamente anidadas.

Aplicaciones en el ámbito empresarial

Se utiliza en la integración de sistemas empresariales, intercambio de datos entre aplicaciones, y representación de información estructurada en la web

Se usa en industrias como finanzas, salud, electrónico. gestion de contenido y comercio
